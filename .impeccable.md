## Design Context

### Users
Students, reviewers, and operators inspecting an urban AQI monitoring prototype for Ho Chi Minh City. They need to compare latest conditions, forecast horizons, model outputs, hotspots, and artifact freshness quickly during demos or analysis sessions.

### Brand Personality
Calm, civic, operational. The interface should feel like a focused city-monitoring console: precise, readable, and trustworthy.

### Aesthetic Direction
Map-first operational dashboard with restrained surfaces, compact controls, clear AQI color semantics, and honest empty or stale states. Use the map as the visual anchor rather than decorative imagery.

### Design Principles
- Put current/forecast state and freshness near the map, not behind explanatory text.
- Make Random Forest vs GBT and H+1..H+24 comparisons direct.
- Show only real metrics and artifacts; missing data should be explicit.
- Keep controls dense but touch-friendly on mobile.
- Preserve accessibility with visible labels, focus states, and reduced-motion behavior.
